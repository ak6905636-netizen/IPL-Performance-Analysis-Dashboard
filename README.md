# 🏏 IPL Performance Analysis Dashboard

## 📌 Project Title

**IPL Performance Analysis Dashboard**

---

## 📖 Description

This project presents an interactive **IPL (Indian Premier League) performance dashboard** built to analyze batting, bowling, and match-level insights. The dashboard helps uncover key trends such as team performance, player contributions, wicket distribution, and scoring patterns.

It is designed for **data-driven decision-making** and provides a clear understanding of match dynamics using visual analytics.

---

## 🎯 Problem Statement

Cricket teams, analysts, and fans often struggle to extract meaningful insights from large volumes of IPL match data. The challenge is to:

* Understand **team-wise performance trends**
* Identify **top-performing batsmen and bowlers**
* Analyze **run distribution and strike rates**
* Examine **wicket patterns and match progression**

This dashboard solves these challenges by converting raw data into **interactive visual insights**.

---

## 🎯 Objectives

* Analyze total runs scored by each team
* Identify top batsmen based on total runs
* Evaluate strike rate performance
* Analyze types of extras (wides, no-balls, etc.)
* Study wicket distribution by dismissal type
* Understand match flow through overs

## 🛠️ Tools & Technologies Used

* **Power BI** – Data Visualization
* **Excel / CSV Dataset** – Data Source
* **DAX** – Calculations & Measures

---

## 📂 Dataset Features

The dataset includes:

* Match details (innings, overs, balls)
* Player information (batsman, bowler)
* Runs scored (batsman runs, extras)
* Wicket details (type, player out)
* Teams and match context

---

## 🚀 How to Use

1. Open the Power BI `.pbix` file
2. Use filters on:

   * Batting Team
   * Bowler
   * Batsman
   * Innings
3. Interact with visuals to explore insights
